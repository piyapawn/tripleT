HINT

This is a tic tac toe game mixed with turn-based style.


No. 1 : Choose your character.

Each character has a different stat. choose the character you like.


No. 2 : Fight

When either side wins that round, it can damage the opponent's HP.


No. 3 : Win

The game is complete when either side has 0 HP remaining.


No. 4 : Draw

At the end of the 10th round, no one has 0 HP left. will be considered a tie and the game is over immediately.
